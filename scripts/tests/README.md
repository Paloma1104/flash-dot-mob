# Flash.Mob Testing Suite

This directory contains comprehensive test scripts for the AP Token Economy implementation.

## Test Files

1. **test_contracts.sh** - Smart contract compilation and unit tests
2. **test_ui.md** - UI/UX test checklist
3. **test_integration.md** - Integration test scenarios
4. **test_economy.md** - Token economy validation

## Running Tests

### Smart Contracts

```bash
cd test_scripts
./test_contracts.sh
```

### Manual UI Testing

Follow the checklists in `test_ui.md`

### Integration Testing

Follow scenarios in `test_integration.md`
