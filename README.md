## 🏃 Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn
- Foundry (for contracts)
- EAS CLI (for mobile builds)

### 1. Clone & Install

```bash
git clone https://github.com/yourname/flash.mob.git
cd flash.mob
./scripts/setup.sh
```

### 2. Configure Environment

```bash
cp .env.example .env
# Edit .env with your API keys:
# - EXPO_PUBLIC_MAPBOX_TOKEN
# - EXPO_PUBLIC_PRIVY_APP_ID (optional)
# - EXPO_PUBLIC_SENTRY_DSN (optional)
```

### 3. Run Development Server

```bash
npm run dev
```

### 4. Build for Device

```bash
# iOS
npm run build:dev:ios

# Android
npm run build:dev:android
```
